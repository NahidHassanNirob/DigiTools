# Digital Tools Buying Website

## Description
A responsive e-commerce website for digital tools where users can browse products, add them to the cart, and proceed to checkout.

## Technology Used
- React.js
- Tailwind CSS & DaisyUI
- JavaScript (ES6+)
- React-Toastify
- React-Icons
- JSON

## Features
1. Dynamic product listing with 3-column layout  
2. Cart functionality with add, remove, and checkout actions  
3. Real-time cart count in the Navbar with notifications
